## Getting started

```bash
First you clone the project:

$ git clone https://github.com/MRLSK8/Pokedex.git && cd Pokedex

Then you run the following command to install packages:

$ yarn install

Next, you start the project:

$ yarn start

Finally you run it like so:

$ yarn android
```
